# Holiday Provider

A lightweight PHP/Symfony bundle that provides public holiday data for Malaysia (Kuala Lumpur) and Hong Kong.

## Features

- Get public holidays by country/state and year
- Check if a given date is a holiday
- Get the next upcoming holiday
- Fully static data
- Symfony bundle compatible

---

## Installation

Install via Composer:

```bash
composer require hisdeveloper/holiday-provider
